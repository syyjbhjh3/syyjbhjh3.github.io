## Overview ⭐️

- static website로 Github pages 활용한 웹 호스팅 가능
- 개발자 맞춤형 포트폴리오 웹 페이지
- 제작자 표기 시, 무료로 활용가능

## Prerequisites 🍪

- PC에 git이 설치되어 있어야하며, Github 계정이 있어야 합니다.
- HTML 문법을 몰라도 텍스트만 교체하여 활용 가능합니다.
- HTML 문법을 알면 레이아웃이나 디자인을 변경할 수 있습니다.
- CSS를 몰라도 제작 가능하지만, CSS에 친숙하다면 디자인도 자유롭게 변경할 수 있습니다.
